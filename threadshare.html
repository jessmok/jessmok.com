ThreadShare

TOOLS: HTML, CSS, React, JavaScript, Firebase

ThreadShare was a collaborative effort to create a basic discussion forum where users can keep up with news and discuss interesting topics with peers. This website currently supports anonymous, authenticated, and administrative users.

To do so, we used Firebase to store thread data and handle authentication features and used React, HTML, JavaScript, and CSS to render the user interface.
